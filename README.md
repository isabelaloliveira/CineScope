# CineScope

Aplicação web para descobrir filmes e séries, com busca por título, navegação por gênero e favoritos. Interface single-page sem backend, consumindo a API do TMDB, com favoritos persistidos via localStorage.

## Funcionalidades

- Busca de filmes e séries por título
- Navegação por gênero
- Seção "Em alta" com os títulos da semana
- Modal de detalhes com sinopse, gêneros, nota e duração
- Favoritos salvos no navegador, persistentes entre sessões
- Tema escuro e layout responsivo

## Tecnologias

- HTML, CSS e JavaScript puro
- API do TMDB
- Google Fonts (Bebas Neue e Inter)
- Sem frameworks ou dependências externas

## Como usar

Abra o arquivo `index.html` diretamente no navegador. Nenhuma instalação necessária.

O projeto já inclui uma chave pública de demonstração da API do TMDB; por ser uma aplicação estática, ela é somente leitura e com limite de requisições. Para usar uma chave própria, substitua o valor em `js/api.js`.

Para publicar online, basta ativar o GitHub Pages no repositório.
